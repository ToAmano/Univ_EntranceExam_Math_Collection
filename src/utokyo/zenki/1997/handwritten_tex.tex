\subsection*{第1問}

\noindent
\textbf{[解]}
(1) $\triangle OAB$ の中点を $M$ とする. $MC = \frac{\sqrt{3}}{2} AB$ 及び $MC \perp AB$ だから, $\vec{AB} = \begin{pmatrix} -a \\ b \end{pmatrix}$ より
\[
\vec{MC} = \pm \frac{\sqrt{3}}{2} \begin{pmatrix} b \\ a \end{pmatrix}
\]
となり,
\[
\vec{OC} = \vec{OM} + \vec{MC} = \frac{1}{2} \begin{pmatrix} a \\ b \end{pmatrix} \pm \frac{\sqrt{3}}{2} \begin{pmatrix} b \\ a \end{pmatrix} = \frac{1}{2} \begin{pmatrix} a \pm \sqrt{3}b \\ b \pm \sqrt{3}a \end{pmatrix} \quad (\text{複号同順})
\]
$C$ は第1象限の点だから, 複号正を採用して, $C\left(\frac{a + \sqrt{3}b}{2}, \frac{b + \sqrt{3}a}{2}\right)$ である. $\triangle ABC$ が $D$ に含まれるには, $A, B, C$ が $D$ に含まれていれば良い. $a, b > 0$ とあわせて,
\[
\begin{cases}
0 < a \le 1, \quad 0 < b \le 1 \\
0 \le a + \sqrt{3}b \le 2, \quad 0 \le b + \sqrt{3}a \le 2
\end{cases} \quad \cdots *
\]
これを図示して, 右図斜線部 (境界含む, が軸除く).

(2) $\triangle ABC$ の一辺は $\sqrt{a^2+b^2}$ だから
\[

\subsection*{第2問}

\noindent
\textbf{[解1]} 
\[
(\text{与式}) \iff \left( \frac{n^2}{2n+1} - m \right) a + m^2 + m > 0 \quad \cdots \text{①}
\]
ここで, $f(m) = -\frac{m^2+m}{A-m} \quad \left(A = \frac{n^2}{2n+1}\right)$ とおく.
\[
f(m) = +\left( m+A+1 + \frac{A(A+1)}{m-A} \right)
\]
\[
f'(m) = +\left( \frac{m^2-2Am+A}{(m-A)^2} \right)
\]
より, 下表を与える.

\begin{center}
\begin{tabular}{c|c|c|c|c|c|c|c}
$m$ & $\dots$ & $-\frac{n}{2n+1}$ & $\dots$ & $A$ & $\dots$ & $n$ & $\dots$ \\ \hline
$f'$ & $+$ & $0$ & $-$ & / & $-$ & $0$ & $+$ \\ \hline
$f$ & $\nearrow$ & $\frac{1}{2n+1}$ & $\searrow$ & / & $\searrow$ & $2n+1$ & $\nearrow$
\end{tabular}
\end{center}

\bigskip

\noindent
①が全ての $m \in \mathbb{Z}$ で成り立つ条件は
\[
\begin{cases}
A - m > 0 \iff A > m \text{ を満たす任意の } m \text{ で } a > f(m) \\
A - m = 0 \text{ の時, } m > 0 \text{ だから ①は必ず成立} \\
A - m < 0 \iff A < m \text{ を満たす任意の } m \text{ で } a < f(m)
\end{cases}
\]
したがって, $y=f(x)$ のグラフから
\[
0 < a < 2n+1
\]

\draw[->] (0,-2.5) -- (0,3.5) node[above] {$y$};

\draw[dashed] (1.2,-2.5) -- (1.2,3.5) node[above] {$A$};

\draw[domain=1.6:3.2, samples=50, smooth, thick] plot (\x, {(\x*\x+\x)/(\x-1.2)});

\node[below] at (-1,0) {$-\frac{n}{2n+1}$};

\bigskip

\noindent
\textbf{[解2]} $f(x) = x^2 - (a-1)x + Aa$ とおく $\left(A = \frac{n^2}{2n+1}\right)$. $f(x)=0$ の判別式 $D$ として
\[
D = (a-1)^2 - 4Aa = a^2 - 2(1+2A)a + 1
\]
から, $D < 0 \iff (1+2A) - \sqrt{4A^2+4A} < a < (1+2A) + \sqrt{4A^2+4A}$
\[
\iff \frac{1}{2n+1} < a < 2n+1
\]
の時, 任意の $x$ に対し $f(x) > 0$ となる. その他の時, $f(x)=0$ は
\[
x = \frac{1}{2}\left[ (a-1) \pm \sqrt{(a-1)^2 - 4Aa} \right] = \frac{1}{2}\left[ (a-1) \pm \sqrt{a^2 - 2(1+2A)a + 1} \right]
\]
を解に持つ. これらを $\alpha, \beta$ ($\alpha \le \beta$) とする.

\bigskip

\noindent
$1^\circ \ 0 < a \le \frac{1}{2n+1}$ の時
\[
\alpha = \frac{1}{2}\left[ (a-1) - \sqrt{a^2-2(1+2A)a+1} \right] > -1
\]
\[
\iff (a+1)^2 > a^2 - 2(1+2A)a + 1 \iff 2(1+A)a > 0 \quad \cdots \text{③}
\]
③は成立するので, $\alpha > -1$. 同様に $\beta < 0$ だから, $-1 < \alpha < \beta < 0$.
したがって任意の $m \in \mathbb{Z}$ に対し $f(m) > 0$.

\bigskip

\noindent
$2^\circ \ 2n+1 \le a$
\[
f(n) = (A-n)a + n^2 + n = \frac{-(n^2+n)}{2n+1} a + n^2 + n \le 0
\]
から不適.

\bigskip

\noindent
以上まとめて $0 < a < 2n+1$.

\subsection*{第3問}

\noindent
\textbf{[解]} $0 < r < 1 \quad \cdots \text{①}$

\bigskip

\noindent
(1) $|\vec{PA}| = |\vec{PB}|$ から, $P$ は線分 $AB$ の垂直2等分面 $x=y$ 上にあるので, $P(p,p,q)$ と表せる. この時
\[
|\vec{PA}| = r |\vec{PO}| \iff |\vec{PA}|^2 = r^2 |\vec{PO}|^2
\]
で,
\[
(p-1)^2 + p^2 + q^2 = r^2 (2p^2 + q^2)
\]
を満たす $p, q$ の存在条件をしらべれば良い.
\[
2(1-r^2)p^2 - 2p + (1-r^2)q^2 + 1 = 0 \quad \cdots \text{②}
\]
\[
2(1-r^2)\left[ p - \frac{1}{2(1-r^2)} \right]^2 + (1-r^2)q^2 + 1 - \frac{1}{2(1-r^2)} = 0
\]
①及び $p, q \in \mathbb{R}$ から, $p, q$ の存在条件は
\[
1 - \frac{1}{2(1-r^2)} \le 0 \iff \frac{\sqrt{2}}{2} \le r < 1 \quad (\because \text{①})
\]

\draw[->] (0,0,0) -- (0,3,0) node[above] {$z$};

\draw[thick] (O) -- (B) node[below right] {$B$};

\draw[thick] (P) circle (1pt) node[above right] {$P$};

\draw (P) -- (B);
\draw (P) -- (O);
\draw[dashed] (P) -- (1.5, 0, 1.5);
\end{tikzpicture}
\end{center}

\bigskip

\noindent
(2) ②が満たされているとする. また $r \to 1-0$ をかんがえるので, $r$ は(1)の条件を満たすとして良い.
\[
\vec{PA} \cdot \vec{PB} = \begin{pmatrix} 1-p \\ -p \\ -q \end{pmatrix} \cdot \begin{pmatrix} -p \\ 1-p \\ -q \end{pmatrix} = 2p(p-1) + q^2
\]
\[
= 2p(p-1) + \left[ \frac{2}{1-r^2}p - \frac{1}{1-r^2} - 2p^2 \right] = \frac{2r^2}{1-r^2}p - \frac{1}{1-r^2} \quad \cdots \text{③}
\]
ここで, ②から
\[
p = \frac{1}{2(1-r^2)}\left[ 1 \pm \sqrt{1 - 2(1-r^2)\left\{(1-r^2)q^2+1\right\}} \right] = \frac{1}{2(1-r^2)}\left[ 1 \pm \sqrt{2r^2-1 - 2(1-r^2)^2 q^2} \right]
\]
これを $\alpha, \beta$ ($\alpha \le \beta$) とすると, ①から
\[
\begin{cases}
\max \beta = \frac{1}{2(1-r^2)} \left[ 1 + \sqrt{2r^2-1} \right] \equiv a \\
\min \alpha = \frac{1}{2(1-r^2)} \left[ 1 - \sqrt{2r^2-1} \right] \equiv b
\end{cases}
\]
で, $b \le p \le a$ である. ②, ③から
\[
M(r) - m(r) = \frac{2r^2}{1-r^2}(a-b) = \frac{2r^2 \cdot 2}{2(1-r^2)^2} \sqrt{2r^2-1}
\]
だから
\[
(1-r)^2 \{ M(r) - m(r) \} = \frac{2(1-r)^2 r^2}{(1+r)^2(1-r)^2} \sqrt{2r^2-1}
\]
\[
= 2 \left(\frac{r}{1+r}\right)^2 \sqrt{2r^2-1} \longrightarrow \frac{1}{2} \quad (r \to 1-0)
\]

\noindent
上のように折り返して考えると, 光線は直線を進む. この時, 各頂点の座標は $\left(l, \frac{\sqrt{3}}{2}(2k)\right), \left(m+\frac{1}{2}, \frac{\sqrt{3}}{2}(2k-1)\right)$ と表せる. 又, $l$ がはじめてこれらの点とぶつかった時の点が, 到達する頂点である.

\bigskip

\noindent
(1) 光線 $l : y = \frac{\sqrt{3}}{4}x$ の時, 頂点は $\left(t, \frac{\sqrt{3}}{4}t\right)$ で表される. これがはじめて $\left(m+\frac{1}{2}, \frac{\sqrt{3}}{2}(2k-1)\right)$ の形になる時, $t = 2p \ (p \in \mathbb{N})$ が必要で, この時, $y$ 座標 $2p$ だから, $\min p = 2$ であり, もとめる頂点は B.

\bigskip

\noindent
(2) $l : y = \frac{\sqrt{3}}{6k+2}x$ の時, 頂点は $\left(t, \frac{\sqrt{3}}{6k+2}t\right)$ で表せる. これが $(m+\frac{1}{2}, \dots)$ の形になる時, $\frac{t}{3k+1} \in \mathbb{N}$ となることが必要で, $t = (3k+1)p \ (p \in \mathbb{N})$ と表せる. この時 $t \in \mathbb{Z}$ だから, $(m+\frac{1}{2}, \dots)$ より $\frac{t}{3k+1} = p$ が偶数であることが必要. これらをみたす $\min p = 2$ より, たどりつく座標は $(6k+2, \sqrt{3})$ である.

また $y$ 座標にある頂点は, その $x$ 座標を3でわって2あまると C だから, もとめる頂点は C. 又, 反射回数は, $l$ と各直線(AB, BC, CA)の共通点の数にひとしい. まず, $0 < y < \frac{\sqrt{3}}{2}$ の時, $(0 \le x < 3k+1), (t \le x < t+1) \ (t \in \mathbb{N})$ の区間で各々2本の直線と交わり, $0 < t \le 1$ の区間では1と交わるから, 計 $6k+1$ コ. 次に $y = \frac{\sqrt{3}}{2}$ の時, $x = 3k+1$ だから, ここで交わる直線は $y = \frac{\sqrt{3}}{2}$ のみで 1コ. 長くり $<\sqrt{3}$ の時も同様にして $6k+1$ コだから, あわせて
\[
12k+3 \text{ コ}
\]
