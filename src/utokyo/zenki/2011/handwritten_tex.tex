\subsection*{第2問}
[解]
(1) $1 < \sqrt{2} < 2$ から、$\langle a \rangle = \sqrt{2}-1$ とする。$a_1 = \sqrt{2}-1$、$a_2 = \langle \frac{1}{\sqrt{2}-1} \rangle = \sqrt{2}-1$ だから、
以下帰納的に $a_n = \sqrt{2}-1$

(2) まず、$a_1 = \langle a \rangle$ から、$\frac{1}{2} \le a < 1$ $\cdots$ ①が必要。
\[ a_{n+1} = \left\langle \frac{1}{a_n} \right\rangle = \left\langle \frac{1}{a} \right\rangle = a \]
より、$m \in \mathbb{Z}_{\ge 0}$ として
\[ \frac{1}{a} - m = a \quad \therefore a^2 + ma - 1 = 0 \]
から $a = \frac{-m \pm \sqrt{m^2+4}}{2}$ だから、①に代入
\[ \frac{2}{3} \le -m \pm \sqrt{m^2+4} < 2 \]
\[ m + \frac{2}{3} \le \pm \sqrt{m^2+4} < m + 2 \]
$m \ge 0$ から、複号正を採用して、この時、各項0以上だから2乗してよく、
\[ \frac{4}{9} m + \frac{4}{9} \le 4 < 4m + 4 \]
\[ \therefore 0 < m \le \frac{8}{3} \]
$m \in \mathbb{Z}_{\ge 0}$ から、$m = 1, 2$ だから、$a = \frac{-1+\sqrt{5}}{2}, -1+\sqrt{2}$

(3) $p$ と $q$ は互いに素であるとする。$a_n \neq 0$ かつ $a_{n+1}=0$ の時 (ii)から $\frac{1}{a_n} \in \mathbb{Z}$ である。
$q=1$ の時は明らかに成立するから、以下 $q \ge 2$ とする。$p$ を $q$ で割った商を $j_1$、あまりを $r_1$ として、
\[ p = j_1 q + r_1 \quad (0 \le r_1 < q) \]
$r_1=0$ の時は $a_1 = 0$ となり題意は成立するから、以下 $r_1 \neq 0$ とする。この時
\[ a_1 = \left\langle \frac{p}{q} \right\rangle = \left\langle j_1 + \frac{r_1}{q} \right\rangle = \frac{r_1}{q} \quad (\neq 0) \]
となる。$q$ を $r_1$ で割った商とあまりを、$j_2, r_2$ とすると
\[ q = j_2 r_1 + r_2 \quad (0 \le r_2 < r_1) \]
\[ \therefore a_2 = \left\langle \frac{1}{a_1} \right\rangle = \left\langle \frac{q}{r_1} \right\rangle = \frac{r_2}{r_1} \]
$r_2=0$ ならば $a_2=0$、$r_2 \neq 0$ ならば $r_1$ を $r_2$ で割った余りを $r_3$ とする。以下同様に、$m \in \mathbb{N}$ に対し、
$r_m = 0$ なら $a_m = 0$、$r_m \neq 0$ なら $r_{m-1}$ を $r_m$ で割ったあまりを $r_{m+1}$ とする。（ただし $r_0 = q$）
ここで、$q$ 以下の自然数 $l$ で、$r_l = 0$ となるものが存在しないと仮定すると、
\[ q > r_1 > r_2 > \dots > r_q \ge 1 \]
なる自然数 $r_1, r_2, \dots, r_q$ が得られ、矛盾。よって、$r_l = 0$ となるものが存在し、この時 $q$ 以上の全ての自然数に対し、$a_n = 0$ となる。

\subsection*{第4問}
[解] $\alpha < \beta \dots$ ①として良い。$PQ^2 = PR^2$ だから、
\begin{align*}
(\beta - \frac{1}{2})^2 + (\beta^2 - \frac{1}{4})^2 &= (\alpha - \frac{1}{2})^2 + (\alpha^2 - \frac{1}{4})^2 \\
\beta^2 - \beta + \beta^4 - \frac{1}{2}\beta^2 &= \alpha^2 - \alpha + \alpha^4 - \frac{1}{2}\alpha^2 \\
\beta^4 + \frac{1}{2}\beta^2 - \beta &= \alpha^4 + \frac{1}{2}\alpha^2 - \alpha \\
(\beta^2 + \alpha^2)(\beta^2 - \alpha^2) + \frac{1}{2}(\beta^2 - \alpha^2) - (\beta - \alpha) &= 0
\end{align*}
$\alpha \neq \beta$ から
\[ (\beta^2 + \alpha^2)(\beta + \alpha) + \frac{1}{2}(\beta + \alpha) - 1 = 0 \quad \cdots \text{①} \]
ここで、$p = \alpha + \beta, \ q = \alpha \beta$ とすると、$\alpha, \beta$ の実数条件から、
\[ p^2 - 4q > 0 \quad \cdots \text{②} \]
である。①に代入して
\[ (p^2 - 2q)p + \frac{1}{2}p = 1 \quad \cdots \text{③} \]
次に、$G$ について
\begin{align*}
X &= \frac{\alpha+\beta+\frac{1}{2}}{3} = \frac{1}{3}(p+\frac{1}{2}) \quad \cdots * \\
Y &= \frac{\alpha^2+\beta^2+\frac{1}{4}}{3} = \frac{1}{3}(p^2-2q+\frac{1}{4})
\end{align*}
である。③において、$p=0$ とすると $0=1$ となり矛盾、つまり $p \neq 0 \dots$ ④ であるから

\subsection*{第5問}
[解] $p, q \in \mathbb{N}$,
\[ -q \le b \le 0 \le a \le p, \quad b \le c \le a, \quad a, b, c \in \mathbb{Z} \quad \cdots \text{①} \]

(1) まず前者についてかんがえる。
\[ W([a, b; c]) = p - q - (a+b) = -q \quad \therefore p = (a+b) \]
$b \le 0, \ a \le p$ から、このようになるには $(a, b) = (p, 0)$ が必要で、①からこの時、
$c = 0, 1, \dots, p$ とすれば十分。よって、$p+1$ コある。

次に後者について、
\[ W([a, b; c]) = p - q - (a+b) = p \quad \therefore -q = (a+b) \]
$-q \le b, \ 0 \le a$ から、$(a, b) = (0, -q)$ が必要で、①から $c = 0, -1, \dots, -q$ とすれば
十分で、$q+1$ コある。

(2) $p = q$ の時、$s \in \mathbb{Z}$ に対して
\[ W([a, b; c]) = -(a+b) = -p + s \quad \therefore a+b = p-s \]
①から、$-p \le a+b \le p$ だから、$0 \le s \le 2p$ である。この時は、$b = p-s-a$ から、
$-p \le p-s-a \le 0 \iff p-s \le a \le 2p-s$ の時のみ、各 $a$ に対して $b$ がただ1つ対応する。
よって、$s$ によって以下のようになる。

(i) $0 \le s \le p$ の時
$p-s \le a \le p$ で、各 $(a,b)$ に対し、$c$ は
\[ a - b + 1 = a - (p-s-a) + 1 = 2a + (1+s-p) \text{ 通り} \]
あるから、全部で
\begin{align*}
\sum_{a=p-s}^{p} \{ 2a + (1+s-p) \} &= \frac{1}{2} \cdot 2 \cdot (2p-s)(s+1) + (1+s-p)(s+1) \\
&= (p+1)(s+1)
\end{align*}

(ii) $p \le s \le 2p$ の時
$0 \le a \le 2p-s$ で (i) と同様に
\begin{align*}
\sum_{a=0}^{2p-s} \{ 2a + (1+s-p) \} &= \frac{1}{2} \cdot 2 \cdot (2p-s)(2p-s+1) + (1+s-p)(2p-s+1) \\
&= (p+1)(2p-s+1)
\end{align*}

以上まとめて、
\[ \begin{cases}
(p+1)(s+1) & (0 \le s \le p) \\
(p+1)(2p-s+1) & (p \le s \le 2p) \\
0 & (\text{otherwise})
\end{cases} \]

(3) $s$ について和をとれば良く、$p \ge 1$ なら
\begin{align*}
&\sum_{s=0}^{p-1} (p+1)(s+1) + \sum_{s=p}^{2p} (p+1)(2p-s+1) \\
&= (p+1) \sum_{s=1}^{p} s + (p+1) \sum_{s=1}^{p+1} s \\
&= (p+1) \left[ \frac{1}{2}p(p+1) + \frac{1}{2}(p+1)(p+2) \right] = (p+1)^3
\end{align*}
