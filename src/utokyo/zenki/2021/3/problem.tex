関数
\[ f(x)=\frac{x}{x^2+3} \]
に対して，$y=f(x)$のグラフを$C$とする。
点$A(1, \, f(1))$における$C$の接線を
\[ l:y=g(x) \]
とする。
\begin{enumerate}
  \item$C$と$l$の共有点で$A$と異なるものがただ1つ存在することを示し，
その点の$x$座標を求めよ。
  \item(1)で求めた共有点の$x$座標を$\alpha$とする。定積分
\[ \int_{\alpha}^1 { \{ f(x)-g(x) \} }^2 dx \]
を計算せよ。
\end{enumerate}
