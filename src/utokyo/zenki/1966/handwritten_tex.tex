\section*{第1問}

\noindent
[\textbf{解}] $a = 0.1\alpha$, $b = 0.1\beta$ とおく $(\alpha, \beta \in \mathbb{N})$。キロリスの運賃を $f(x)$ とする。
\[
\begin{cases}
f(319) = 970 \\
f(349) = 1010
\end{cases}
\quad \dots \star
\]
であり、後半の文章から、
\begin{itemize}
    \item $f(319)$ は $300a + 19b = 30\alpha + 1.9\beta$ の10円未満を切り上げたもの
    \item $f(349)$ は $300a + 49b = 30\alpha + 4.9\beta$ の 〃
\end{itemize}
でもある。したがって、
\[
\begin{cases}
960 < 30\alpha + 1.9\beta \le 970 \\
1000 < 30\alpha + 4.9\beta \le 1010
\end{cases}
\quad \dots \text{①}
\]
①々引いて、
\[
30 < 3\beta < 50 \quad \therefore 10 < \beta < 16.6\dots
\]
だから $\beta \in \mathbb{N}$ とあわせて、$\beta = 11, 12, 13, 14, 15, 16$ である。以下 $A = f(349) - f(319)$ とする。\\
$A = 40$ となることが必要である。又、$30\alpha$ は10の倍数である。

\begin{enumerate}
    \item[$1^\circ$] $\beta = 11$\\
    $1.9\beta = 20.9$, $4.9\beta = 53.9$ より、$A = 60 - 30 = 30$ で不適。
    
    \item[$2^\circ$] $\beta = 12$\\
    $1.9\beta = 22.8$, $4.9\beta = 58.8$ より $A = 60 - 30 = 30$ 〃
    
    \item[$3^\circ$] $\beta = 13$\\
    $1.9\beta = 24.7$, $4.9\beta = 63.7$ より $A = 70 - 30 = 40$ だが、$\star$に代入して
    \[
    970 = 30\alpha + 30 \quad \therefore \alpha = \frac{94}{3} \notin \mathbb{N}
    \]
    で不適。
    
    \item[$4^\circ$] $\beta = 14$\\
    $1.9\beta = 26.6$, $4.9\beta = 68.6$ より $3^\circ$と同じく不適。
    
    \item[$5^\circ$] $\beta = 15$\\
    $1.9\beta = 28.5$ より $3^\circ$と同じく不適。
    
    \item[$6^\circ$] $\beta = 16$\\
    $1.9\beta = 30.4$, $4.9\beta = 78.4$ より、$\star$に代入して $\alpha = 31$ を得て十分。
\end{enumerate}
以上から、$(\alpha, \beta) = (31, 16)$、つまり $(a, b) = (3.1, 1.6)$ である。

\section*{第2問}

\noindent
[\textbf{解}] まず、$l$ が $y$ 軸平行の時、$x = k\ (k \in \mathbb{R})$ とおける。この時 $(X, Y) = (4x + 2y, x + 3y)$ とおくと、
\[
X = 4k + 2y
\]
となり、題意をみたさず不適。そこで、$l: y = mx + n$ とおける $(m, n \in \mathbb{R})$。この時、
\[
\begin{cases}
X = 4x + 2(mx + n) = (4 + 2m)x + 2n \\
Y = x + 3(mx + n) = (1 + 3m)x + 3n
\end{cases}
\]
で表される点 $(X, Y)$ が再び $l$ 上にあるので、
\[
(1 + 3m)x + 3n = m\{(4 + 2m)x + 2n\} + n
\]
が $x$ についての恒等式で、
\[
\begin{cases}
1 + 3m = m(4 + 2m) \\
3n = 2mn + n
\end{cases}
\quad \therefore (m, n) = \left(\frac{1}{2}, 0\right), (-1, 0)
\]
だから、$l: y = \frac{1}{2}x,\ y = -x$ である。

\bigskip
\noindent
[\textbf{解2 - 略}] $y = f(x)$ とおく。$x = 0$ での成立が必要で、$(2f(0), 3f(0))$ も $l$ 上にある。そこで $f(0) = 0$ か否かで場合分けして答えが求まる。

\section*{第3問}

\noindent
[\textbf{解}] $x$ 軸を実軸、$y$ 軸を虚軸とするガウス平面を考える。\\
また、$e(\theta) = \cos\theta + i\sin\theta$ とおく。$\overrightarrow{P_{n-1}P_n}$ の表す複素数を $\alpha_n\ (n \in \mathbb{N})$ とすると、
\[
\begin{cases}
\alpha_{n+1} = 2 e\left(\frac{2}{3}\pi\right) \alpha_n \\
\alpha_1 = 1
\end{cases}
\]
だから、くり返し用いて
\[
\alpha_n = 2^{n-1} \cdot e^{n-1}\left(\frac{2}{3}\pi\right)
\]
であるから、$\overrightarrow{OP_{3n}}$ の表す複素数 $P$ として
\begin{align*}
P &= \sum_{k=1}^{3n} \alpha_k \\
&= \frac{1 - 2^{3n} e^{3n}\left(\frac{2}{3}\pi\right)}{1 - 2 e\left(\frac{2}{3}\pi\right)} \\
&= \frac{1 - 2^{3n} e(2n\pi)}{1 - 2 \left(-\frac{1}{2} + \frac{\sqrt{3}}{2}i\right)} \\
&= \frac{2 + \sqrt{3}i}{7} \left(1 - 2^{3n}\right)
\end{align*}
だから、
\[
P_{3n} \left( \frac{2}{7}(1 - 2^{3n}), \frac{\sqrt{3}}{7}(1 - 2^{3n}) \right)
\]
である。

\section*{第4問}

\noindent
[\textbf{解}] 方程式の解は、$xy$ 平面での $y = \frac{1}{x}$, $y = \sin\frac{\pi x}{6}$ の交点の $x$ 座標に他ならない。これらのグラフの概形は右図。

\begin{center}
\begin{tikzpicture}[scale=0.9, >=stealth]
    \draw[->] (-0.5,0) -- (7,0) node[right] {$x$};
    \draw[->] (0,-0.5) -- (0,1.5) node[above] {$y$};
    \node[below left] at (0,0) {$O$};
    
    \draw[thick, domain=0:6.2, samples=100] plot (\x, {sin(\x*180/6)}) node[above right] {$y = \sin\frac{\pi x}{6}$};
    \draw[thick, domain=0.7:6.5, samples=100] plot (\x, {1/\x}) node[above right] {$y = \frac{1}{x}$};
    
    \draw[dashed] (3,0) -- (3,1);
    \node[below] at (3,0) {$3$};
    \draw[dashed] (6,0) -- (6,0);
    \node[below] at (6,0) {$6$};
\end{tikzpicture}
\end{center}

\noindent
まず、$6 \le x$ の時、方程式が解を持たないことを示す。
\[
\begin{cases}
6 \le x \le 12 \text{ の時 } \sin\frac{\pi x}{6} \le 0 < \frac{1}{x} \text{ から明らか。} \\
12 \le x \text{ の時 } \frac{1}{x} \le \frac{1}{12} > 1 \ge \sin\frac{\pi x}{6} \text{ から明らか}
\end{cases}
\]
（別注: $f(x) = \sin\frac{\pi x}{6} - \frac{1}{x}$ とすると、$f(3) = \frac{2}{3} > 0$, $f(6) = -\frac{1}{6} < 0$ から $3 < x < 6$ に解を持つ）

\medskip
\noindent
以上から、与式の最大の解は $3 < x < 6$ にある。$f(x)$ は同区間で単調減少で
\begin{align*}
f(3) &= \frac{2}{3}, \quad f(4) = \frac{\sqrt{3}}{2} - \frac{1}{4} \ ( > 0) \\
f(4.5) &= \frac{\sqrt{2}}{2} - \frac{1}{4.5} \ ( > 0), \quad f(5) = -\frac{1}{18} \ ( < 0)
\end{align*}
だから、$4.5 < x < 5$ にもとめる解があるので、最も近い整数月は $5$ である。

\section*{第5問}

\noindent
[\textbf{解}] 時刻 $0$ で $OX$ を $x$ 軸正方向とし、$OX$ が $x$ 軸から $y$ 軸正方向へと回転する座標をおく。この時、時刻 $t$ での $P$ の座標 $(X, Y)$ として

\begin{center}
\begin{tikzpicture}[scale=1.2, >=stealth]
    \draw[->] (-0.3,0) -- (2.5,0) node[right] {$x$};
    \draw[->] (0,-0.3) -- (0,2.0) node[above] {$y$};
    \node[below left] at (0,0) {$O$};
    
    \coordinate (P) at (1.8, 1.2);
    \draw[thick] (0,0) -- (P) node[above right] {$P$};
    \fill (P) circle (1.5pt);
    
    \draw[->] (0.6,0) arc (0:33.69:0.6);
    \node at (0.8,0.2) {$t$};
\end{tikzpicture}
\end{center}

\[
\begin{cases}
X = e^{2t} \cos t \\
Y = e^{2t} \sin t
\end{cases}
\]
だから
\[
\begin{cases}
X' = e^{2t} (-\sin t + 2\cos t) \\
Y' = e^{2t} (\cos t + 2\sin t)
\end{cases}
\]
となり、もとめる長さ $L$ として ($P$ は $0 \le t \le 2\pi$ では重ならないので)
\begin{align*}
L &= \int_0^{2\pi} \sqrt{(X')^2 + (Y')^2} \, dt \\
&= \int_0^{2\pi} \sqrt{5} e^{2t} \, dt \\
&= \frac{\sqrt{5}}{2} \left[ e^{2t} \right]_0^{2\pi} \\
&= \frac{\sqrt{5}}{2} \left( e^{4\pi} - 1 \right)
\end{align*}

\section*{第6問}

\noindent
[\textbf{解}] 4ケタの数のつくり方は ${}_9\mathrm{P}_4$ 通りで同様にたしかしい。このうち、$1966$ より小さいのは、以下の2つである。
\[
\begin{cases}
1^\circ\ 1900\text{未満} \\
2^\circ\ 1900\text{以上 } 1965\text{以下}
\end{cases}
\]

\noindent
$1^\circ$ となるのは
\[
\underset{\begin{subarray}{c} \uparrow \\ \text{4位} \end{subarray}}{1} \times \underset{\begin{subarray}{c} \uparrow \\ \text{3位} \end{subarray}}{7} \times 7 \times 6 = 6 \times 7^2 \text{ 通り} \quad \dots \text{①}
\]
である。一方、$2^\circ$ となるのは、
\[
\underbrace{4 \times 6}_{1960\text{未満}} + \underbrace{4}_{1960\text{以上}} = 28 \text{ 通り} \quad \dots \text{②}
\]
①, ②から、求める確率 $P$ は
\[
P = \frac{6 \times 7^2 + 28}{{}_9\mathrm{P}_4} = \frac{322}{9 \cdot 8 \cdot 7 \cdot 6} = \frac{161}{1512} = \frac{23}{216}
\]
