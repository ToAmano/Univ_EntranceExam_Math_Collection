$n\geqq3$ とし，
正 $n$ 角錐の表面を，
底面に含まれない $n$ 個の辺で切り開いて得られる展開図を考える．
正 $n$ 角錐の頂点は，
展開図においては，
異なる $n$ 個の点になっている．
ここでは，
これら $n$ 個の点を通る円の半径が1であるような，
正 $n$ 角錐のみを考えることにする．

\begin{figure}[htbp]

  \begin{tikzpicture}[scale=0.9]

      \draw[thick] (P) -- (A1);
      \draw[thick] (P) -- (A2);
      \draw[thick] (P) -- (A3);

    \begin{scope}[shift={(2.5, 0)}]
      \foreach \i in {1,...,5} {

      }
      \draw[dashed] (B1) -- (B2) -- (B3) -- (B4) -- (B5) -- cycle;

      \draw[thick] (B1) -- (V1) -- (B2) -- (V2) -- (B3) -- (V3) -- (B4) -- (V4) -- (B5) -- (V5) -- cycle;
    \end{scope}
  \end{tikzpicture}
\end{figure}

\begin{enumerate}
  \item 各 $n$ に対して，
  このような正 $n$ 角錐の体積の最大値 $v_n$ を求めよ．
  \item $\displaystyle\lim_{n\to\infty}v_n$ を求めよ．
\end{enumerate}
