[解 1] $A(t, 0), B(-t, 0) \; (t > 0)$ となるよう座標平面をとる。$P(x, y)$ とする。
$$ \overrightarrow{PA} = \begin{pmatrix} t - x \\ -y \end{pmatrix}, \quad \overrightarrow{PB} = \begin{pmatrix} -t - x \\ -y \end{pmatrix} $$
だから、
$$ |\overrightarrow{PA}| = \sqrt{(t - x)^2 + y^2}, \quad |\overrightarrow{PB}| = \sqrt{(t + x)^2 + y^2}, \quad \overrightarrow{PA} \cdot \overrightarrow{PB} = -(t^2 - x^2) + y^2 $$
となるので、与式に代入して、
\begin{align*}
& \sqrt{(t - x)^2 + y^2} \sqrt{(t + x)^2 + y^2} = c + t^2 - (x^2 + y^2) \\
\iff & \left\{ (t - x)^2 + y^2 \right\} \left\{ (t + x)^2 + y^2 \right\} = \left\{ c + t^2 - (x^2 + y^2) \right\}^2 \\
\iff & \begin{cases} \frac{x^2}{t^2 + \frac{1}{2} c} + \frac{y^2}{\frac{1}{2} c} = 1 & (\because c > 0, t > 0) \\ c + t^2 \ge x^2 + y^2 \end{cases}
\end{align*}
で、これを図示すると右図。よって、もとめる軌跡は
$$ \frac{x^2}{t^2 + \frac{1}{2} c} + \frac{y^2}{\frac{1}{2} c} = 1 $$

[解 2] $\overline{AB} = 2t, \overline{PA} = a, \overline{PB} = b, \angle APB = \theta$ とする。
$\triangle ABP$ が一直線上にない時、$\triangle ABP$ に余弦定理を用いて、
$$ 4t^2 = a^2 + b^2 - 2ab \cos \theta \cdots \text{①} $$
これは $A, B, P$ が一直線上の時 ($\theta = 0$) にも成立する。又、与式から、
$$ ab (1 + \cos \theta) = c $$
$$ \therefore ab \cos \theta = c - ab $$
①及び $-1 \le \cos \theta \le 1$ に代入して、
\begin{align*}
& 4t^2 = a^2 + b^2 - 2(c - ab) = (a + b)^2 - 2c \\
\iff & \begin{cases} a + b = \sqrt{4t^2 + 2c} & \cdots \text{②} \\ c \le 2ab & (\because a, b, c \ge 0) \cdots \text{③} \end{cases}
\end{align*}

②が [解 1] の座標平面で表すのは、原点を中心、長半径 $\frac{1}{2}(a+b) = \sqrt{t^2 + \frac{1}{2} c}$, 短半径 $\sqrt{t^2 + \frac{1}{2} c - t^2} = \sqrt{\frac{1}{2} c}$ の
楕円: $\frac{x^2}{t^2 + \frac{1}{2} c} + \frac{y^2}{\frac{1}{2} c} = 1$ である。この時 $a+b=A$ とすると $\frac{A}{2} - t \le a \le \frac{A}{2} + t$ で
$$ ab = a(A - a) \ge \left( \frac{A}{2} - t \right) \left( \frac{A}{2} + t \right) = \frac{A^2}{4} - t^2 = \frac{1}{2} c \quad (\text{等号成立は } a = \frac{A}{2} \pm t) $$
で、③は常に成立。したがって、もとめる軌跡は
$$ \frac{x^2}{t^2 + \frac{1}{2} c} + \frac{y^2}{\frac{1}{2} c} = 1 $$
となる。
