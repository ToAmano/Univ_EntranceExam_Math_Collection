[解] (1) $a_n^2 = A_n$ などと表す。与式の分母を払って $(A_0, A_1, B_0, B_1 > 0)$
$$ B_1 (B_0 + 1) + A_1 (A_0 + 1) > A_1 (B_0 + 1) + B_1 (A_0 + 1) $$
を示せば良い。
\begin{align*}
(\text{左辺}) - (\text{右辺}) &= B_1 (B_0 - A_0) + A_1 (A_0 - B_0) \\
&= (B_1 - A_1) (B_0 - A_0) > 0 \quad (\because A_0 < B_0, A_1 < B_1)
\end{align*}
よって示された。

(2) $\{x_k\}$ は自然数列 $\{k\}$ の並びかえである。(1) から、$\frac{x_i^2}{i^2+1} + \frac{x_j^2}{j^2+1} > \frac{x_j^2}{i^2+1} + \frac{x_i^2}{j^2+1} \; (i < j \in \mathbb{N})$ に対し $x_i < x_j$ なら
だから、$T(n) = \sum_{k=1}^n \frac{x_k^2}{k^2+1}$ の値は、$x_i$ と $x_j$ を入れかえた方が小さくなる。したがって、
$k = 1, 2, \cdots$ の順に $x_k$ と $x_m$ $(m = k+1, \cdots, n)$ の大きさを比べ、$x_k < x_m$ ならそのまま、$x_k > x_m$ なる $m$ があるならそのうち $x_m$ が最小のものと $x_k$ を入れかえる作業をすることで、「$x_k = k$」の時、$T(n)$ が $\min$ であることがわかる。
$$ T(n) \ge \sum_{k=1}^n \frac{k^2}{k^2+1} = \sum_{k=1}^n \left( 1 - \frac{1}{k^2+1} \right) = n - \sum_{k=1}^n \frac{1}{k^2+1} \cdots \text{①} $$

ここで、$y = \frac{1}{x^2+1}$ のグラフから、面積比較して、
$$ \sum_{k=1}^n \frac{1}{k^2+1} \le \frac{1}{2} + \int_1^n \frac{1}{x^2+1} dx $$

$\tan \alpha_n = n$ なる $\alpha_n$ を定めると $(0 \le \alpha_n < \pi/2)$
$$ \sum_{k=1}^n \frac{1}{k^2+1} \le \frac{1}{2} + \alpha_n - \pi/4 $$
①から
$$ T(n) \ge n - \frac{1}{2} - \alpha_n + \pi/4 > n - \frac{1}{2} - \pi/4 \quad (\because \alpha_n < \pi/2) $$
ここで、$\pi < 3.15 \cdots$ だから、
$$ T(n) > n - \frac{1}{2} - \frac{3.15}{4} > n - \frac{5}{4} $$
$\left( 32 > 25.75 \text{ から } \frac{5.15}{4} < \frac{5}{4} \right)$

(別解: $\sum \frac{1}{k^2+1}$ の評価)
\begin{align*}
\sum_{k=1}^n \frac{1}{k^2+1} &< \sum_{k=2}^n \frac{1}{k^2-1} + \frac{1}{2} = \sum_{k=2}^n \frac{1}{2} \left( \frac{1}{k-1} - \frac{1}{k+1} \right) + \frac{1}{2} \\
&= \frac{1}{2} + \frac{1}{2} \left( 1 + \frac{1}{2} - \frac{1}{n} - \frac{1}{n+1} \right) \\
&= \frac{5}{4} - \frac{1}{2} \left( \frac{1}{n} + \frac{1}{n+1} \right) < \frac{5}{4}
\end{align*}
