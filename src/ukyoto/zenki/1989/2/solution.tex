[解2] $A_n = \sum_{i=1}^n a_i$ とおく。$i=1,2,\cdots,n$ に対し
\[ a_i \le \frac{A_n - a_i}{n-1} \cdots \text{①} \]
だから、$i$ についてたして、$n-1 > 0$ から
\[ (n-1)A_n \le (n-1)A_n \]
より、等号が成立する。つまり、①の等号が全て成立するからセイリして
\[ a_i = A_n / n \]
であるから
\[ a_1 = a_2 = \cdots = a_n \quad \text{//} \]
の時、題意がみたされる。
