\subsection*{第1問}
[解] $\angle A_n O B_n = \theta_n$ とおくと、題意から
帰納的に
\[ \theta_n = \left(\frac{1}{2}\right)^{n-1} \theta \cdots \text{①} \]
右下図で、
\[ OA_n = a_n \]
\[ OB_{n+1} = a_{n+1} \]
だから
\[ \begin{cases} a_n \cos\left(\left(\frac{1}{2}\right)^n \theta\right) = a_{n+1} \\ a_1 = 1 \end{cases} \]
となり、
\[ a_n = \cos\left\{ \left(\frac{1}{2}\right)^{n-1} \theta \right\} \cdots \cos\frac{\theta}{8} \cos\frac{\theta}{4} \cos\frac{\theta}{2} \]
である

(1)
\[ a_3 \cdot \sin\frac{\theta}{4} = \sin\frac{\theta}{4} \cdot \cos\frac{\theta}{4} \cdot \cos\frac{\theta}{2} = \frac{1}{2} \sin\frac{\theta}{2} \cdot \cos\frac{\theta}{2} = \frac{1}{4} \sin\theta \quad \text{//} \]

(2) $a_n \sin\frac{\theta}{2^{n-1}} = A_n$ も、(1)と同じように計算して
\[ A_n = \left(\frac{1}{2}\right)^{n-1} \sin\theta \]
だから
\[ a_n = \frac{ \left(\frac{1}{2}\right)^{n-1} \theta }{ \sin\left\{ \left(\frac{1}{2}\right)^{n-1} \theta \right\} } \cdot \frac{\sin\theta}{\theta} \to \frac{\sin\theta}{\theta} \quad (n \to \infty) \quad \text{//} \]

\subsection*{第2問}
[解2] $A_n = \sum_{i=1}^n a_i$ とおく。$i=1,2,\cdots,n$ に対し
\[ a_i \le \frac{A_n - a_i}{n-1} \cdots \text{①} \]
だから、$i$ についてたして、$n-1 > 0$ から
\[ (n-1)A_n \le (n-1)A_n \]
より、等号が成立する。つまり、①の等号が全て成立するからセイリして
\[ a_i = A_n / n \]
であるから
\[ a_1 = a_2 = \cdots = a_n \quad \text{//} \]
の時、題意がみたされる。

\subsection*{第3問}
[解] $f(x)$ はモニック多項式であるとしてよく、$f(x) = x^3 + ax^2 + bx + c$ とおける。
$f'(x) = 3x^2 + 2ax + b$ より、
\[ f(x) = \frac{1}{3}\left(x + \frac{1}{3}a\right)f'(x) + \left(\frac{2}{3}b - \frac{2}{9}a^2\right)x + (c - ab) \]
題意から、
\[ \frac{2}{3}b - \frac{2}{9}a^2 = 0 \quad \therefore \quad b = \frac{1}{3}a^2 \]
だから
\[ f(x) = x^3 + ax^2 + \frac{1}{3}a^2 x + c \]
\[ f'(x) = 3x^2 + 2ax + \frac{1}{3}a^2 = 3\left(x + \frac{1}{3}a\right)^2 \ge 0 \]
より、$f(x)$ は単調増加。これと $f(x) \to \pm\infty \ (x \to \pm\infty)$ から、$f(x) = 0$ をみたす実数 $x$ がただ1つある。($f$ は連続) //

\subsection*{第5問}
[解] 立方体の6面をA, B, C, D, E, Fとする。Aにぬる色を固定すると、対面も同じ色でぬらなければならない。のこりの側面の色のぬり方は、
2色 $\alpha, \beta$ とすると Cにどちらをぬるかの2通りである。以上から
\[ P(N) = \left(\frac{1}{N}\right)^5 {}_{N-1}P_2 = \frac{(N-1)(N-2)}{N^5} \quad \text{//} \]
さて、$x = \frac{1}{N} \ (0 < x \le \frac{1}{3})$ とすると、
\[ P(N) = x^3(1-x)(1-2x) \]
\[ \frac{dP}{dx} = 10x^4 - 12x^3 + 3x^2 = 10x^2 \left(x - \frac{6+\sqrt{6}}{10}\right)\left(x - \frac{6-\sqrt{6}}{10}\right) \]
で、$0 < x \le \frac{1}{3}$ より、$\frac{dP}{dx} > 0$ つまり $P$ は $x$ について単調増加だから、$N$ について単調減少。
したがって、
\[ \begin{cases} a < b \text{ のとき } P(a) > P(b) \\ a > b \text{ のとき } P(b) > P(a) \end{cases} \quad \text{//} \]

\subsection*{第6問}
[解] 高さ $y$ の時の表面の面積 $S$、円錐の半径を $r$ とすると
\[ 10 - y = S \frac{dy}{dt} \cdots \text{①} \]
$\therefore S = \pi \left( \frac{r(10-y)}{10} \right)^2$ を代入してセイリ
\[ dt = \frac{\pi r^2}{100}(10-y)dy \]
両辺積分して
\[ t = \frac{\pi r^2}{100}\left(10y - \frac{1}{2}y^2\right) + C \]
$t=0$ で $y=0$ から $C=0$。$(t, y) = (540, 2), y=10$ を各々代入
\[ \begin{cases} 540 = \frac{\pi r^2}{100} \cdot 18 \\ t = \frac{\pi r^2}{100} \cdot 50 \end{cases} \]
辺々わって
\[ \frac{t}{540} = \frac{50}{18} \cdot 54 \]
\[ \therefore t = 1500 \]
だから、タンクがいっぱいになるのは、あと
\[ \frac{1500}{60} - 9 \text{ (h)} = 16 \text{ (h)} \]
後である。
