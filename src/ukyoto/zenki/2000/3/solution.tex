(1) $\vec{c} = \begin{pmatrix} p \\ q \\ r \end{pmatrix}$ とおく。($p^2+q^2+r^2=1$, $p,q,r \in \mathbb{R}$)。$|\vec{a}|=|\vec{b}|=|\vec{c}|=1$ から、$\alpha, \beta$ は各々 $\vec{a}$ と $\vec{c}$, $\vec{b}$ と $\vec{c}$ のなす角にひとしい。
\begin{align*}
\cos\alpha &= \begin{pmatrix} 1 \\ 0 \\ 0 \end{pmatrix} \cdot \begin{pmatrix} p \\ q \\ r \end{pmatrix} = p \\
\cos\beta &= \begin{pmatrix} 1/2 \\ \sqrt{3}/2 \\ 0 \end{pmatrix} \cdot \begin{pmatrix} p \\ q \\ r \end{pmatrix} = \frac{1}{2}p + \frac{\sqrt{3}}{2}q
\end{align*}
だから、
\begin{align*}
(与式左辺) &= p^2 - \frac{1}{2}p^2 - \frac{\sqrt{3}}{2}pq + \frac{1}{4}(p+\sqrt{3}q)^2 \\
&= \frac{1}{2}p^2 - \frac{\sqrt{3}}{2}pq + \frac{1}{4}p^2 + \frac{\sqrt{3}}{2}pq + \frac{3}{4}q^2 \\
&= \frac{3}{4}(p^2+q^2) \le \frac{3}{4}(p^2+q^2+r^2) = \frac{3}{4} \ (\because r \in \mathbb{R}, =)
\end{align*}

(2) $t = \cos\alpha, s = \cos\beta$ とおく。(1) から
\begin{align*}
&s^2 - ts + t^2 - \frac{3}{4} \le 0 \\
&\frac{1}{2}(t - \sqrt{3}| \sin\alpha |) \le s \le \frac{1}{2}(t + \sqrt{3}| \sin\alpha |)
\end{align*}
$0 \le \alpha \le \pi$ から $\sin\alpha \ge 0$ だから
\begin{align*}
&\frac{1}{2}(\cos\alpha - \sqrt{3}\sin\alpha) \le \cos\beta \le (\cos\alpha + \sqrt{3}\sin\alpha)/2 \\
&\cos\left(\alpha + \frac{\pi}{3}\right) \le \cos\beta \le \cos\left(\alpha - \frac{\pi}{3}\right)
\end{align*}
$0 \le \beta \le \pi$ では $\cos\beta$ は単調減少だから

$1^\circ$ $0 \le \alpha \le \frac{\pi}{3}$ の時
\[ \frac{\pi}{3} - \alpha \le \beta \le \alpha + \frac{\pi}{3} \]

$2^\circ$ $\frac{\pi}{3} \le \alpha \le \frac{2}{3}\pi$ の時
\[ \alpha + \frac{\pi}{3} \ge \beta \ge \alpha - \frac{\pi}{3} \]

$3^\circ$ $\frac{2}{3}\pi \le \alpha \le \pi$ の時
\[ \alpha - \frac{\pi}{3} \le \beta \le -\alpha + \frac{5}{3}\pi \]
