[解] $p \leftarrow prime, \ a, b \in \mathbb{N} \ (a \perp b)$ $\cdots$ ①
$A = (a+bi)^p$ とおく。
\[ A = (a^p - pC_2 a^{p-2}b^2 + \cdots) + i(pC_1 a^{p-1}b - pC_3 a^{p-3}b^3 + \cdots) \]
①から、$A$ の虚部は $p$ の倍数である。これを $B$ とおく。
\begin{align*}
B &= pC_1 a^{p-1}b - pC_3 a^{p-3}b^3 + pC_5 a^{p-5}b^5 - \cdots \\
&= \sum_{k \in \mathbb{N}}^{2k-1 \le p} (-1)^{k-1} pC_{2k-1} a^{p-(2k-1)} b^{2k-1}
\end{align*}
$A \in \mathbb{R}$、つまり $B=0$ と仮定する。

$1^\circ$ $p=2$ の時
\[ A = a^2 - b^2 + 2abi \text{ より } B = 2ab \neq 0 \text{ で矛盾 } (\because \text{①}) \]

$2^\circ$ $p \ge 3$ の時
$p$ は奇数だから
\[ B = (a\text{の倍数}) \pm b^p = 0 \]
の形でかけるが、これは $b$ が $a$ の倍数であることを表し、$a \perp b$ に矛盾。

以上から示された。(Q.E.D)
