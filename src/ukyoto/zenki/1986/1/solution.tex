[解] 
$\begin{cases}
a_1 \le a_2 \le \cdots \le a_n & \cdots \text{①} \\
a_1 + a_2 + \cdots + a_n = 0 & \cdots \text{②}
\end{cases}$

①, ②から、
$n a_1 \le a_1 + a_2 + \cdots + a_n \le n a_n$ \\
$\therefore a_1 \le 0 \le a_n \quad (\because n \in \mathbb{N})$ \\
より、$a_k \le 0 \le a_{k+1} \ (k=1, 2, \cdots, n-1)$ をみたす$k$が存在する

\begin{align*}
\sum_{i=1}^{n} i a_i &= (a_1 + \cdots + a_n) + (a_2 + \cdots + a_n) + \cdots + (a_n) \\
&= 0 + (-a_1) + (-a_1 - a_2) + \cdots + (-a_1 - \cdots - a_k) \\
&\quad + (a_{k+1} + \cdots + a_n) + \cdots + a_n \\
&> 0 \quad (\because \text{各項0以上、等号成立は} a_1 = a_2 = \cdots = a_n = 0 \text{だが、題意よりそれはない})
\end{align*}

[解2] (カンタンに$\dots$)
M.I.で示す。$n=k$での成立をかていし、$a_1, a_2, \cdots, (a_k + a_{k+1})$に適用すると
\begin{align*}
0 &< \sum_{i=1}^{k} i a_i + k a_{k+1} \\
&\le \sum_{i=1}^{k+1} i a_i \quad (\because a_{k+1} \ge 0) \quad \text{終}
\end{align*}
