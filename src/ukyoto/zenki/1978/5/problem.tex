$m$，$n$は整数とし，$f(x)=x^3+mx^2+nx+2$とする．
\begin{enumerate}
  \item 方程式$f(x)=0$が三つの整数解(重解があってもよい)をもつような$m$，$n$の組をすべて求めよ．
  \item 方程式$f(x)=0$が少なくとも一つの整数解をもつために$m$，$n$がみたすべき必要十分条件を求めよ．
  \item $|m|\leqq5$，$|n|\leqq5$の範囲で，(ii)の条件をみたす$m$，$n$の組はいく通りあるか．
\end{enumerate}
