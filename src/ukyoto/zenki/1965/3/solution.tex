[\textbf{解}]
\begin{align}
&-\frac{\pi}{2} \le \alpha, \beta \le \frac{\pi}{2} \dots \text{①} \\
&\sin(x+\alpha) + \sin(x+\beta) = \sqrt{3}\sin x \dots \text{②}
\end{align}
②が $x$ についての恒等式なので、$x = 0, \frac{\pi}{2}$ での成立が必要で
\begin{align}
\sin\alpha + \sin\beta &= 0 \dots \text{③} \\
\cos\alpha + \cos\beta &= \sqrt{3} \dots \text{④}
\end{align}
①, ③から $\beta = -\alpha \dots \text{⑤}$ である。④に代入して
\[
2\cos\alpha = \sqrt{3} \iff \cos\alpha = \frac{\sqrt{3}}{2} \iff \alpha = \pm\frac{\pi}{6} \quad (\text{∵ } \text{①})
\]
だから、
\[
(\alpha, \beta) = \left( \pm\frac{\pi}{6}, \mp\frac{\pi}{6} \right) \quad (\text{複号同順})
\]
が必要。逆にこの時、
\[
(\text{与左辺}) = 2\sin x \cos\frac{\pi}{6} = \sqrt{3}\sin x = (\text{右辺})
\]
となって分（十分）。よって
\[
(\alpha, \beta) = \left( \pm\frac{\pi}{6}, \mp\frac{\pi}{6} \right) \quad (\text{複号同順})
\]
