[解] $n \in \mathbb{Z}_{\ge 0}$ である
(1) 漸化式から、法を3として $a_{n+2} \equiv a_{n+1} + a_n$ だから、
\begin{align*}
&b_0 = 1,\ b_1 = 2,\ b_2 = 0,\ b_3 = 2,\ b_4 = 2,\ b_5 = 1,\ b_6 = 0, \\
&b_7 = 1,\ b_8 = 1,\ b_9 = 2
\end{align*}

(2) $C_n$ の定義から
$$C_{n+8} - C_n = \underbrace{b_{n+1} + b_{n+2} + b_{n+3} + b_{n+4} + b_{n+5} + b_{n+6} + b_{n+7} + b_{n+8}}_{A}$$
又、(1)から $\{b_n\}$ は $\{1, 2, 0, 2, 2, 1, 0, 1\}$ の周期数列だから、
$A$ の中にはこれらの要素が1つずつ入っているので、
$$A = \sum_{k=1}^8 b_k = C_8$$
だから
$$C_{n+8} = C_n + C_8 \quad \square$$

(3) $C_8 = 9$ より $C_{n+8} = C_n + 9$ である。以下、題意を帰納的に示す。\\
$1^\circ\ n=0\sim 7$ の時、
\begin{align*}
1 \le C_0 = 1 \le \frac{3}{2} && 5 \le C_4 = 7 \le 9 \\

$2^\circ\ n=8k, 8k+1, \cdots, 8k+7$ ($k \in \mathbb{Z}_{\ge 0}$) での成立を仮定\\

$$n+1+9 \le C_{n+8} \le \frac{3}{2}(n+1)+9$$
$$n+9 \le C_{n+8} \le \frac{3}{2}(n+7) \le \frac{3}{2}(n+9)$$
