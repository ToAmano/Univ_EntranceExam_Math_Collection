[解] 正四面体の一辺の長さを1として良い。座標空間において、
$$A\left(0, 0, \frac{\sqrt{6}}{3}\right),\ B\left(\frac{\sqrt{3}}{3}, 0, 0\right)$$
$$C\left(-\frac{\sqrt{3}}{6}, \frac{1}{2}, 0\right),\ D\left(-\frac{\sqrt{3}}{6}, -\frac{1}{2}, 0\right)$$
とおける。すると $B, C, D$ を通る円は $z=0, x^2+y^2 = \frac{1}{3}$ である。

$xz$平面で立体を切断すると左下図で線分$FG$の中点を$M$とし、
$EM$と$xy$平面の交点を$N$、$AM$と$x$軸の交点を$H$とすると、メネラウスの定理から
$$\frac{1-s}{s} \cdot \frac{NH}{BN} \cdot \frac{t}{1-t} = 1$$
$$BN = \frac{1-s}{s} \frac{t}{1-t} NH$$
だから $NH = p$ として、
$$\left(\frac{\sqrt{3}}{2} + p\right) = \frac{1-s}{s} \frac{t}{1-t} p$$
$$\therefore p = \frac{\frac{\sqrt{3}}{2}}{\frac{1-s}{s}\frac{t}{1-t} - 1} \quad (>0)$$
だから、$N\left(-\frac{\sqrt{3}}{6} - p, 0, 0\right)$ である。図形の対称性から、条件は
$$-\frac{\sqrt{3}}{6} - p \ge -\frac{\sqrt{3}}{3} \quad (\because ON \le \text{円の半径})$$
$$\Leftrightarrow 4s(1-t) \le (1-s)t \quad \left(\because p>0 \text{から} \frac{1-s}{s}\frac{t}{1-t} > 1 \Leftrightarrow t > s\right)$$
これと $0 < t, s < 1$ を図示して、下図（境界は $t = \frac{4s}{1+3s}$ のみ含む）
