\noindent
[\textbf{解}] 題意から $y = f'(a)(x-a) + f(a)$ が $(c,0)$ を通る.
\[
0 = f'(a)(c-a) + f(a) \quad \dots \text{①}
\]
又, $g(x)$ の定義から
\[
g'(x) = \frac{f'(x)(x-c) - f(x)}{(x-c)^2}
\]
だから
\[
g'(a) = \frac{f'(a)(a-c) - f(a)}{(a-c)^2} = 0 \quad (\because \text{①}) \quad \mathbin{/\mkern-5mu/}
\]
