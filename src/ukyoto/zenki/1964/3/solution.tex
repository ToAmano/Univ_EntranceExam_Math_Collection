(イ) $\vec{AP} = p \vec{c}$ と定めると $\vec{a}, \vec{c}$ では次の通り。
$\vec{a}, \vec{c}$ を用いて表して
\[
\begin{cases}
\vec{Q} = k [(1-p)\vec{a} + p\vec{c}] = (1-t)\vec{a} + t \frac{1}{3}\vec{c} \\
\vec{R} = K [(1-p)\vec{a} + p\vec{c}] = (1-t')\vec{a} + t' \frac{2}{3}\vec{c}
\end{cases}
\]
これをといて、$k = \frac{1}{2p+1}$, $K = \frac{2}{p+2}$ だから
\[
\begin{aligned}
\overline{BQ} : \overline{QR} : \overline{RP} &= \frac{1}{2p+1} : \frac{2}{p+2} - \frac{1}{2p+1} : 1 - \frac{2}{p+2} \\
&= p+2 : 3p : p(2p+1)
\end{aligned}
\]

\[
p(2p+1) \leqq 3p \leqq p+2

\[
\therefore \overline{RP} \leqq \overline{QR} \leqq \overline{BQ} \quad (\text{等号成立は } p=0 \text{ 又は } p=1 (\text{この時一括右は成立}))
\]

\begin{aligned}
a < b+c &\iff p+2 < 2p^2 + 4p \\
