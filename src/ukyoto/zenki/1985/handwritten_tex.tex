\subsection*{第1問}
[解] $AB=b, AC=c$ とおく。\\
($b, c > 0, q < b+c$) と、余弦定理から
\[ q^2 = b^2 + c^2 - 2p \]
$b+c = \alpha, bc = \beta$ とおくと
\[
\begin{cases}
q < \alpha & \cdots \text{①} \\
\alpha^2 - 2\beta = 2p + q^2 & \cdots \text{②}
\end{cases}
\]
で、$b, c$ は $t$ の2次式 $t^2 - \alpha t + \beta = 0$ の2正実解である。以下、$b, c$ の存在条件をもとめる。まず、この判別式を $D$ として $b, c > 0$ から
\[
\begin{cases}
D \ge 0 \\
\alpha > 0 \\
\beta > 0
\end{cases} \iff
\begin{cases}
\alpha^2 - 4\beta \ge 0 & \cdots \text{③} \\
\alpha > 0 & \cdots \text{④} \\
\beta > 0 & \cdots \text{⑤}
\end{cases}
\]
②から $\beta = \frac{1}{2}(\alpha^2 - 2p - q^2)$ だから、③⑤に代入して、
\[
\begin{cases}
\alpha^2 - 2(\alpha^2 - 2p - q^2) \ge 0 & \cdots \text{⑥} \\
\alpha^2 - 2p - q^2 > 0 & \cdots \text{⑦}
\end{cases}
\]
$q>0$ から、①$\wedge$④$\wedge$⑦をみたす $\alpha$ の存在条件をもとめればよいことになる。\\
⑥$\wedge$⑦ $\iff 2p + q^2 < \alpha^2 \le 4p + 2q^2 \quad \cdots \text{⑧}$\\
より $2p + q^2 < 4p + 2q^2 \iff 2p + q^2 > 0 \cdots \text{⑨}$が必要（この時 $4p + 2q^2 > 0$ で十分）。この時各辺正から、\\
⑧ $\iff \sqrt{2p + q^2} < \alpha \le \sqrt{4p + 2q^2} \quad \cdots \text{⑩}$\\
で、①、⑩をみたす $\alpha$ が存在する条件は
\[ q < \sqrt{4p + 2q^2} \iff 0 < 4p + q^2 \quad (\because \text{両辺正}) \quad \cdots \text{⑪} \]
以上からもとめる条件は⑨$\wedge$⑪で、
\[ 4p + q^2 > 0 \quad (4p + q^2 > 0 \text{の時 } 2p + q^2 > 0) \quad \star \]
逆にこの時、$b^2 + c^2 = 2p + q^2$ となる $b, c \in \mathbb{R}_{>0}$ をとることができて、\\
($\because 2p + q^2 > 0$) $\alpha = b+c$, $\beta = bc$ とおくと
\[ \alpha^2 - 2\beta = 2p + q^2 \]
$b, c$ は $t$ の2次式 $t^2 - \alpha t + \beta = 0$ の2正実解で、$\alpha^2 - 4\beta \ge 0, \alpha, \beta > 0$\\
だから代入して
\[
\begin{cases}
\alpha^2 - 2(\alpha^2 - 2p - q^2) \ge 0 \\
\alpha > 0 \\
\alpha^2 - 2p - q^2 > 0
\end{cases}
\]
$\therefore 2p + q^2 < \alpha^2 \le 4p + 2q^2$\\
$4p + 2q^2 - q^2 > 0$ から $q^2 < 4p + 2q^2$ なので、$\alpha = b+c > q$ をみたすような $(b, c)$ が存在し十分。\\
以上から
\[ 4p + q^2 > 0 \quad \text{//} \]

[別解] [多変数でのベクトルの強さ、、]\\
\fbox{\parbox{\textwidth}{B, Cを固定し、Aを動かして考える。この時Aがあるような$p, q$をもとめると考えて、$\overrightarrow{a}$についてとく。点の存在条件をベクトルのそれに換算するというのが大切。}}\\
点Xに対し、$\overrightarrow{OX} = \vec{x}$ とおく。\\
② $\iff (\vec{b} - \vec{a}) \cdot (\vec{c} - \vec{a}) = p$\\

\subsection*{第3問}
[解]\\
(1) $b, c \in \mathbb{R}$ であるから、この時、$\beta = \overline{\alpha}$ となるので、
\[
\begin{cases}
\alpha + \beta = -2b \quad (\neq 0) \\
\alpha - \beta = 2id \quad (d \in \mathbb{R})
\end{cases}
\]
とおける。したがって、$\gamma = v + iw$ とすると ($v, w \in \mathbb{R}$)\\
$\gamma = \frac{v}{2b}(\alpha + \beta) + \frac{w}{2d}(\alpha - \beta) = \left( -\frac{v}{2b} + \frac{w}{2d} \right) \alpha + \left( -\frac{v}{2b} - \frac{w}{2d} \right) \beta$\\
と表せる。$\blacksquare$

(2) (1)から、$b^2 - c \ge 0$ 又は $b = 0$ が必要である。\\
ア) $b^2 - c \ge 0$ の時\\
$\alpha, \beta$ は実数である。$\alpha = \beta = 0 \iff b = c = 0$ に注意する。
\begin{itemize}
\item $b = c = 0$ の時、$f(x) = 5 \neq 0$ より $(\ast)$ をみたす。
\item otherwise、$t\alpha + u\beta$ は任意の実数値をとるから、
常に $f(x) \neq 0 \iff (b-1)^2 - 5 + c < 0 \iff c < -(b-1)^2 + 5$
\end{itemize}
イ) $b = 0$ かつ $b^2 - c < 0$ の時\\
$\alpha, \beta$ は純虚数であるから、$t\alpha + u\beta$ は任意の $ki$ ($k \in \mathbb{R}$) の形の値をとる。\\
$f(ki) = (-k^2 - c + 5) - 2ki = 0 \iff k = 0 \text{ かつ } c = 5$\\
より、$c \neq 5$ であれば $(\ast)$ をみたす。

ア)、イ) より、もとめるのは、\\
"$b^2 - c \ge 0$ かつ $c < -(b-1)^2 + 5$"\\
or\\
"$b = 0$ かつ $b^2 - c < 0$ かつ $c \neq 5$"\\
で、図示すると右図斜線部(境界は実線のみ含む)

\subsection*{第4問}
[解]\\
(1) ①の中心 $(0,0,0)$ と②のキョリは
\[ l = \frac{r}{\sqrt{1+1+1}} = \frac{\sqrt{3}}{3}r \]
だから、$D$ は半径 $\sqrt{\frac{1}{3}(r^2+2) - \left(\frac{\sqrt{3}}{3}r\right)^2} = \sqrt{\frac{2}{3}}$ の円である。よって、
\[ |PQ| \le (D\text{の直径}) = 2\sqrt{\frac{2}{3}} \quad \blacksquare \]

(2) $x, y, z \in \mathbb{Z}$ から、$\frac{1}{3}(r^2+2) \in \mathbb{Z}$ が必要である。合同式の法を$3$として、
\[
\begin{cases}
r \equiv 0 \text{ の時 } & r^2 + 2 \equiv 2 \\
r \equiv \pm 1 \text{ の時 } & r^2 + 2 \equiv 0
\end{cases}
\]
だから、$r \equiv \pm 1$ が必要である。 $\quad \cdots \text{①}$\\
③を①に代入して $r$ を消す。\\
$x^2 + y^2 + z^2 = \frac{1}{3} \left\{ (x+y+z)^2 + 2 \right\}$\\
$\therefore (x-y)^2 + (y-z)^2 + (z-x)^2 = 2 \quad \cdots \text{②}$\\
$x, y, z \in \mathbb{Z}$ だから、$(x-y)^2, (y-z)^2, (z-x)^2$ のうち2つが1、のこり1つが0である。\\
$\left\{ (x-y)^2, (y-z)^2, (z-x)^2 \right\} = \{1, 1, 0\}, \{1, 0, 1\}, \{0, 1, 1\}$\\
対称性から $x \le y \le z$ とすると、（$\cdots \text{③}$） これをみたすのは\\
$(x, y, z) = (x, x+1, x+1), (x, x, x+1)$\\
のみである。したがって、条件①及び③から、$k \in \mathbb{Z}$ として
\[
\begin{cases}
r = 3k+2 \text{ の時 } & (x, y, z) = (k, k+1, k+1), (k+1, k, k+1), (k+1, k+1, k) \\
r = 3k+1 \text{ の時 } & (x, y, z) = (k, k, k+1), (k, k+1, k), (k+1, k, k)
\end{cases}
\]

\subsection*{第5問}
[解] 2枚の硬貨A, Bとし、表の出る確率が各々 $a, b$ だとする。\\
確率変数 $X_1, X_2$ を以下のように定める。
\[
X_1 = \begin{cases} 0 & (A\text{がウラ}) \\ 1 & (A\text{が表}) \end{cases} \quad X_2 = \begin{cases} 0 & (B\text{がウラ}) \\ 1 & (B\text{が表}) \end{cases}
\]
すると、$X = X_1 + X_2$ で、$X_1, X_2$ は独立だから、
\[
\begin{cases}
P(X=0) = P(X_1=0) \cdot P(X_2=0) \\
P(X=1) = P(X_1=0) \cdot P(X_2=1) + P(X_1=1) \cdot P(X_2=0) \\
P(X=2) = P(X_1=1) \cdot P(X_2=1)
\end{cases} \quad \cdots \text{①}
\]
である。$P(X_1=0) = 1-a, P(X_1=1) = a$ などを①に代入して
\[
\begin{cases}
P(X=0) = (1-a)(1-b) \\
P(X=1) = (1-a)b + (1-b)a \\
P(X=2) = ab
\end{cases} \quad \cdots \text{②}
\]

(1) $P(X=k) = {}_2\mathrm{C}_k \cdot p^k (1-p)^{2-k}$ の時、②に代入して、
\[
\begin{cases}
(1-p)^2 = (1-a)(1-b) = 1 + ab - (a+b) \\
2p(1-p) = a + b - 2ab \\
p^2 = ab
\end{cases}
\]
$\alpha = a+b, \beta = ab$ として、
\[
\begin{cases}
\beta - \alpha = (1-p)^2 - 1 = p(p-2) \\
-2\beta + \alpha = 2p(1-p) \\
\beta = p^2
\end{cases}
\]
第3式を1, 2式に代入して\\
$\alpha = p^2 - p(p-2) = 2p$\\
$\alpha = 2p(1-p) + 2\beta = 2p$\\
から、$a, b$ は $x$ の2次式 $x^2 - 2px + p^2 = 0$ の2解で、$(a, b) = (p, p) \quad \text{//} $

(2) $P(X=k) = \frac{1}{3}$ の時、②に代入して
\[ \frac{1}{3} = 1 + \beta - \alpha = \alpha - 2\beta = \beta \]
よって、$(\alpha, \beta) = \left(1, \frac{1}{3}\right)$ だから、$a, b$ は $t$ の2次式 $t^2 - t + \frac{1}{3} = 0$ の2解だが、この判別式を $D$ として
\[ D = 1 - \frac{4}{3} = -\frac{1}{3} < 0 \]
から、$a, b \in \mathbb{R}$ に反し矛盾。よって $(a, b)$ は存在しない。$\text{//} $

\subsection*{第6問}
[解] (2)で、$x=a$として、
\[ a f(a) = 0 \quad \cdots \text{①} \]
又、(1)から、(2)の両辺 $x$ で微分できて、
\[ b f(x) = f(x) + x f'(x) \]
\[ \therefore (b-1) f(x) = x f'(x) \quad \cdots \text{②} \]
$y = f(x)$ として②に代入すると、
\[ (b-1) y = x \frac{dy}{dx} \]
まず $y \neq 0$ とする。
\[ \frac{b-1}{x} dx = \frac{1}{y} dy \]
両辺積分して、$C_0$ を定数として、
\[ (b-1) \log x = \log y + C_0 \]
\[ \therefore y = C \cdot x^{b-1} \quad (C\text{：定数}) \quad \cdots \text{③} \]
ここで、$y=0$ の時もこの表式で良い。①から、
\[ a \cdot C \cdot a^{b-1} = 0 \iff C = 0 \text{ or } a = 0 \]
に注意すると、$C$ を定数として、
\[ y = C x^{b-1} \quad \text{//} \]
である。\\
{[解注]}\\
さらにこまかく分類すれば、（(1)に注意して）
\[
\begin{cases}
a=0 \wedge 1 \le b \text{ の時 } & y = C x^{b-1} \\
a \neq 0 \text{ or } b < 1 \text{ の時 } & y = 0
\end{cases}
\]
となる。
