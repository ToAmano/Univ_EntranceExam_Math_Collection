[解] $y = -2x^2+x+1$ 上の $x=t$ における接線 $l$ は
\[
l : y = f(x) = (-4t+1)x + 2t^2+1
\]
である. $l$ と $y = x^2$ の交点の $x$ 座標は
\[
x^2 - (1-4t)x - 2t^2 - 1 = 0 \quad \dots \text{①}
\]
の2解で, ①の判別式 $D$ として
\[
D = (1-4t)^2 + 4(2t^2+1) > 0
\]
から, ①は必ず2異実解を持つので, これを $\alpha, \beta \; (\alpha < \beta)$ とすると求める面積は,
\[
S = \int_{\alpha}^{\beta} \{ -x^2 + (1-4t)x + 2t^2+1 \} dx = \frac{1}{6}(\beta-\alpha)^3 \quad \dots \text{②}
\]
ここで, $\alpha < \beta$ ひこうから
\[
\beta - \alpha = \sqrt{(1-4t)^2 + 4(2t^2+1)} = \sqrt{24t^2-8t+5} = \sqrt{24\left(t - \frac{4}{24}\right)^2 + 5 - \frac{16}{24}}
\]
で, これは $t = \frac{1}{6}$ で $\text{min} \;\; \sqrt{\frac{13}{3}}$ をとるから, ②もこの時, $\text{min} \;\; S = \frac{1}{6} \frac{13}{3} \sqrt{\frac{13}{3}} = \frac{13}{18} \sqrt{\frac{13}{3}}$ をとる.
