[解] 対称性から まず $0 \leqq x \leqq \pi/2 \dots$ ①でかんがえる. この時,
\[
\sin^2 x + \cos^2 x = 1
\]
$n \geqq 3$の時
\begin{equation}
\sin^2 x \cdot \sin^{n-2} x + \cos^2 x \cdot \cos^{n-2} x = 1 \quad \dots \text{②}
\end{equation}
$\sin x \neq 1, \cos x \neq 1$の時, $\sin^2 x + \cos^2 x = 1$とあわせて, ②を満たす$x$はないので, $x = 0 \text{ or } \pi/2$である. 対称性から $x = \frac{n\pi}{2} \; (n \in \mathbb{Z})$ である. $n=2$の時は $x$は任意である.

$n=1$の時,
\[
\sqrt{2}\sin(x+\pi/4) = 1 \quad \sin(x+\pi/4) = \frac{\sqrt{2}}{2} \quad \therefore x = 0, \pi/2 \; (\text{in } \text{①})
\]
したがって, 対称性から, $x = \frac{n\pi}{2} \; (n \in \mathbb{Z})$

以上あわせて
\[
\begin{cases}
n \neq 2 \text{ の時 } x = \frac{n\pi}{2} \; (n \in \mathbb{Z}) \\
n = 2 \text{ ,, } \text{任意}
\end{cases}
\]
