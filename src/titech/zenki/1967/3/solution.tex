[解] 題意の方程式の解は $x = \frac{1}{2}\left[ (C_{n+1}+C_{n-1}) \pm \sqrt{(C_{n+1}+C_{n-1})^2 - 4C_{n+1}C_{n-1} + 4{C_n}^2} \right]$
\[
= \frac{1}{2}\left[ (C_{n+1}+C_{n-1}) \pm \sqrt{(C_{n+1}-C_{n-1})^2 + 4(C_{n+1}-C_{n-1})^2} \right] = \frac{1}{2}\left[ (C_{n+1}+C_{n-1}) \pm \sqrt{5}(C_{n+1}-C_{n-1}) \right] \quad \dots \text{①}
\]
である ($\because C_{n+1} \ge C_{n-1}$). $p = \frac{1+\sqrt{5}}{2}, q = \frac{1-\sqrt{5}}{2}$ とおく.

(1) $\alpha_n = p C_{n+1} + q C_{n-1} = p (C_{n+2}-C_n) + q(C_{n+1}-C_n)$
\begin{align*}
&= p(C_{n+2}-C_n) + q(C_{n+2}-2C_n) = (p+q)C_{n+2} - (p+2q)C_n \\
&= C_{n+2} - \frac{3-\sqrt{5}}{2} C_n
\end{align*}
だから
\[
p\alpha_n = p C_{n+2} - \frac{1+\sqrt{5}}{2} \cdot \frac{3-\sqrt{5}}{2} C_n = p C_{n+2} - q C_n = \alpha_{n+1}
\]
\[
\therefore \frac{\alpha_{n+1}}{\alpha_n} = p = \frac{1+\sqrt{5}}{2}
\]

\begin{align*}
\beta_n &= q C_{n+1} + p C_{n-1} = q (C_{n+2}-C_n) + p (C_{n+2}-2C_n) \\
&= C_{n+2} - (2p+q) C_n
\end{align*}
だから
\[
q \cdot \beta_n = q C_{n+2} + p C_n = \beta_{n+1} \quad \therefore \frac{\beta_{n+1}}{\beta_n} = q = \frac{1-\sqrt{5}}{2}
\]

(2) $\alpha_1 = p C_2 + q C_0 = p, \quad \beta_1 = q C_2 + p C_0 = q$ だから くり返し用いて,
\[
\begin{cases}
\alpha_n = p^n = \left( \frac{1+\sqrt{5}}{2} \right)^n \\
\beta_n = q^n = \left( \frac{1-\sqrt{5}}{2} \right)^n
\end{cases}
\]
