[解] $\sin\alpha x \sin\beta x = -\frac{1}{2} \{ \cos(\alpha+\beta)x - \cos(\alpha-\beta)x \}$ であり, $\alpha+\beta \neq 0, \alpha-\beta \neq 0$ から,
\[
\int_0^1 \sin\alpha x \cdot \sin\beta x \, dx = -\frac{1}{2} \left[ \frac{1}{\alpha+\beta}\sin(\alpha+\beta)x - \frac{1}{\alpha-\beta}\sin(\alpha-\beta)x \right]_0^1
\]
\[
= -\frac{1}{2} \left( \frac{1}{\alpha+\beta}\sin(\alpha+\beta) - \frac{1}{\alpha-\beta}\sin(\alpha-\beta) \right) \quad \dots \text{①}
\]
ここで, $\tan\alpha = 2\alpha$ の時 $\cos\alpha = \pm \frac{1}{\sqrt{1+\tan^2\alpha}} = \pm \frac{1}{\sqrt{1+4\alpha^2}}, \quad \sin\alpha = \pm \frac{2\alpha}{\sqrt{1+4\alpha^2}} \quad (\because \text{同号})$
であり, $\alpha > 0$ から $\tan\alpha > 0$, つまり $\sin\alpha, \alpha$ の符号は一致するから
\[
\sin(\alpha+\beta) = \sin\alpha\cos\beta + \sin\beta\cos\alpha = \frac{A}{\sqrt{(1+4\alpha^2)(1+4\beta^2)}}
\]
\[
\sin(\alpha-\beta) = \sin\alpha\cos\beta - \sin\beta\cos\alpha = \frac{B}{\sqrt{(1+4\alpha^2)(1+4\beta^2)}}
\]
ただし $A, B$ は $\sin\alpha$ などの複号において
\[
(A, B) = (2\alpha+2\beta, 2\alpha-2\beta) \quad \text{または} \quad (-2\alpha-2\beta, -2\alpha+2\beta)
\]
をとる. ①に代入して
\[
(\text{与式}) = -\frac{1}{2} \frac{1}{\sqrt{(1+4\alpha^2)(1+4\beta^2)}} \left\{ \frac{A}{\alpha+\beta} - \frac{B}{\alpha-\beta} \right\} = 0
\]
