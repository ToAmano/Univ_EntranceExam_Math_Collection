\begin{enumerate}
  \item$p(x)$，$q(x)$を$x$の3次以下の多項式とする．
すべての$x$に対して$p(x)\cos x=q(x)\sin x$が成り立つならば，
$p(x)$，$q(x)$は恒等的に0に等しいことを示せ．
  \item$P(x)$，$Q(x)$を$x$の3次以下の多項式とする．
すべての$x$に対して
\[ P(x)\cos x+\int_0^xQ(t)\sin tdt=(x^2+2x+3)\sin x \]
が成り立つとき，$P(x)$，$Q(x)$を求めよ．
\end{enumerate}
