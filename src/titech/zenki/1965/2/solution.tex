[解]
\[
\begin{cases}
\alpha + \beta + \gamma = 0 \\
\beta\gamma + \gamma\alpha + \alpha\beta = a \quad \dots \text{①} \\
\alpha\beta\gamma = -b
\end{cases}
\]
である。ここで、$k = \alpha, \beta, \gamma$ に対し、$k^3 + ak + b = 0$ だから
\[
\begin{cases}
k^4 = k(-ak-b) = -ak^2 - bk \\
k^5 = k^2(-ak-b) = -ak^3 - bk^2 = -a(-ak-b) - bk^2 = -bk^2 + a^2 k + ab
\end{cases}
\]
だから
\begin{align*}
a t_5 + b t_4 &= b(-a t_2 - b t_1) + a \{-b t_2 + a^2 t_1 + 3ab\} \\
&= -2ab t_2 + (a^3 - b^2) t_1 + 3a^2 b \quad \dots \text{②}
\end{align*}
ここで
\begin{align*}
t_2 &= (\alpha + \beta + \gamma)^2 - 2(\beta\gamma + \gamma\alpha + \alpha\beta) = -2a \\
t_1 &= 0
\end{align*}
だから、②に代入して
\begin{align*}
(\text{与式}) &= 4a^2 b + 0 + 3a^2 b \\
&= 7a^2 b
\end{align*}
