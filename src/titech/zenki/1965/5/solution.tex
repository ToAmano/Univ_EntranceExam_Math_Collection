[解] 題意から、$0 \le x \le \pi$ に $\sin(x+h) = \sin x$ をみたす $x$ が $1$ つあるので、それを $\alpha$ とする。
つまり
\[
\sin(\alpha+h) - \sin\alpha = 0, \quad 0 \le \alpha \le \pi \quad \dots \text{①}
\]
この時
\[
\int_0^\pi |\sin(x+h) - \sin x| \, dx = \int_0^\alpha \{ \sin(x+h) - \sin x \} \, dx - \int_\alpha^\pi \{ \sin(x+h) - \sin x \} \, dx \quad \dots \text{②}
\]
$F(x) = \cos x - \cos(x+h)$ とおくと、
\begin{align*}
\text{②} &= 2F(\alpha) - F(0) - F(\pi) = 2F(\alpha) - (1 - \cos h) - (-1 + \cos h) \\
&= 2F(\alpha) \quad \dots \text{③}
\end{align*}
ここで $0 < h < \pi$ から、$2\alpha + h = \pi \implies \alpha = \frac{\pi - h}{2}$ だから
\[
F(\alpha) = \cos\left(\frac{\pi}{2} - \frac{h}{2}\right) - \cos\left(\frac{\pi}{2} - \frac{h}{2} + h\right) = 2\sin\frac{h}{2}
\]
だから、③から
\[
\int_0^\pi |\sin(x+h) - \sin x| \, dx < 4\sin\frac{h}{2} < 2h \quad (\because 0 < x \text{ の時 } \sin x < x) \quad \text{證}
\]
