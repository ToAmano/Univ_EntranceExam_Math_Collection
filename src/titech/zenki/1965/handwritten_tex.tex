\section*{第2問}

[解]
\[
\begin{cases}
\alpha + \beta + \gamma = 0 \\
\beta\gamma + \gamma\alpha + \alpha\beta = a \quad \dots \text{①} \\
\alpha\beta\gamma = -b
\end{cases}
\]
である。ここで、$k = \alpha, \beta, \gamma$ に対し、$k^3 + ak + b = 0$ だから
\[
\begin{cases}
k^4 = k(-ak-b) = -ak^2 - bk \\
k^5 = k^2(-ak-b) = -ak^3 - bk^2 = -a(-ak-b) - bk^2 = -bk^2 + a^2 k + ab
\end{cases}
\]
だから
\begin{align*}
a t_5 + b t_4 &= b(-a t_2 - b t_1) + a \{-b t_2 + a^2 t_1 + 3ab\} \\
&= -2ab t_2 + (a^3 - b^2) t_1 + 3a^2 b \quad \dots \text{②}
\end{align*}
ここで
\begin{align*}
t_2 &= (\alpha + \beta + \gamma)^2 - 2(\beta\gamma + \gamma\alpha + \alpha\beta) = -2a \\
t_1 &= 0
\end{align*}
だから、②に代入して
\begin{align*}
(\text{与式}) &= 4a^2 b + 0 + 3a^2 b \\
&= 7a^2 b_{\mathbin{/\mkern-6mu/}}
\end{align*}

\section*{第3問}

[解] $0 \le x < \pi, 0 \le y < \pi \dots \text{①} \quad \alpha = x + y, \beta = x - y$ とおくと条件から
\[
\sin\alpha \cos\beta = \frac{1}{2} \quad \dots \text{②}
\]
$x = \frac{\alpha+\beta}{2}, y = \frac{\alpha-\beta}{2}$ を①に代入して
\[
\begin{cases}
0 \le \alpha + \beta < 2\pi \\
0 \le \alpha - \beta < 2\pi
\end{cases}
\quad \dots \text{③}
\]
③を図示すると右図で、この対称性および $\cos(-\beta) = \cos\beta$ から、$\beta \ge 0$ で考えれば良い。この時
\[
0 \le \beta < \pi, \quad \beta \le \alpha < -\beta + 2\pi \quad \dots \text{④}
\]
まず、$|\beta|$ の最小値について考える。等号成立時、$0 \le \alpha < 2\pi$ で、$\alpha = \frac{\pi}{6}, \frac{5}{6}\pi$ とすれば②をみたす。
この時 $(x, y) = (\frac{\pi}{12}, \frac{\pi}{12}), (\frac{5}{12}\pi, \frac{5}{12}\pi)$ である $\dots \text{⑤}$

次に、$|\beta|$ の最大値について考える。④の区間内では
\[
\begin{cases}
0 \le \beta \le \frac{\pi}{2} \text{ の時} \quad \frac{1}{2} < \sin\alpha \le 1 \quad \dots \text{⑥} \\
\frac{\pi}{2} \le \beta < \pi \text{ の時} \quad -\sin\beta < \sin\alpha \le \sin\beta \quad \dots \text{⑦}
\end{cases}
\]
となる。⑥の時 $\cos\beta = \frac{1}{2} \therefore \beta = \frac{\pi}{3}, \alpha = \frac{\pi}{2}$ の時 $|\beta|$ は最大である。 $\dots \text{⑧}$

⑦の時 $\sin\alpha \cos\beta$ の値域は
\[
\frac{1}{2}\sin 2\beta \le \sin\alpha \cos\beta < -\frac{1}{2}\sin 2\beta
\]
だから②から
\[
\sin 2\beta \le 1 < -\sin 2\beta
\]
これを満たす $\beta$ は存在しない。 $\dots \text{⑨}$

⑧, ⑨および対称性から、$|\beta|$ が最大の時 $(\alpha, \beta) = (\frac{\pi}{2}, \pm \frac{\pi}{3})$ でこの時
\[
(x, y) = (\frac{5}{12}\pi, \frac{1}{12}\pi), (\frac{1}{12}\pi, \frac{5}{12}\pi) \text{ である。} \quad \dots \text{⑩}
\]
⑤, ⑩から、もとめるのは
\[
(x, y) = \left(\frac{1}{12}\pi, \frac{1}{12}\pi\right), \left(\frac{5}{12}\pi, \frac{5}{12}\pi\right), \left(\frac{1}{12}\pi, \frac{5}{12}\pi\right), \left(\frac{5}{12}\pi, \frac{1}{12}\pi\right)_{\mathbin{/\mkern-6mu/}}
\]

\section*{第5問}

[解] 題意から、$0 \le x \le \pi$ に $\sin(x+h) = \sin x$ をみたす $x$ が $1$ つあるので、それを $\alpha$ とする。
つまり
\[
\sin(\alpha+h) - \sin\alpha = 0, \quad 0 \le \alpha \le \pi \quad \dots \text{①}
\]
この時
\[
\int_0^\pi |\sin(x+h) - \sin x| \, dx = \int_0^\alpha \{ \sin(x+h) - \sin x \} \, dx - \int_\alpha^\pi \{ \sin(x+h) - \sin x \} \, dx \quad \dots \text{②}
\]
$F(x) = \cos x - \cos(x+h)$ とおくと、
\begin{align*}
\text{②} &= 2F(\alpha) - F(0) - F(\pi) = 2F(\alpha) - (1 - \cos h) - (-1 + \cos h) \\
&= 2F(\alpha) \quad \dots \text{③}
\end{align*}
ここで $0 < h < \pi$ から、$2\alpha + h = \pi \implies \alpha = \frac{\pi - h}{2}$ だから
\[
F(\alpha) = \cos\left(\frac{\pi}{2} - \frac{h}{2}\right) - \cos\left(\frac{\pi}{2} - \frac{h}{2} + h\right) = 2\sin\frac{h}{2}
\]
だから、③から
\[
\int_0^\pi |\sin(x+h) - \sin x| \, dx < 4\sin\frac{h}{2} < 2h \quad (\because 0 < x \text{ の時 } \sin x < x) \quad \text{證}
\]

\section*{第6問}

[解] 2接点 $P(\alpha, \alpha^2), Q(\beta, \beta^2)$ とおく。$(\alpha < \beta)$。$P, Q$ での接線は
\[
y = 2\alpha x - \alpha^2, \quad y = 2\beta x - \beta^2
\]
だから、この交点は $\left( \frac{\alpha+\beta}{2}, \alpha\beta \right)$ である。これが $(t, -1)$ に等しいので
\[
\begin{cases}
\alpha + \beta = 2t \\
\alpha\beta = -1
\end{cases}
\quad \dots \text{①}
\]
又、題意の面積 $S(t)$ は
\begin{align*}
S(t) &= \int_\alpha^t (x^2 - 2\alpha x + \alpha^2) \, dx + \int_t^\beta (x^2 - 2\beta x + \beta^2) \, dx \\
&= \frac{1}{12} (\beta - \alpha)^3 \quad \dots \text{②}
\end{align*}
であり、①から $\alpha, \beta$ は $x$ の2次方程式 $x^2 - 2tx - 1 = 0$ の2実解で $\alpha < \beta$ とあわせて
\[
\beta - \alpha = 2\sqrt{t^2 + 1}
\]
だから、②とあわせて
\[
\frac{S(t)}{\sqrt{t}} = \frac{1}{12} \frac{8(t^2+1)\sqrt{t^2+1}}{\sqrt{t}} = \frac{2}{3} \sqrt{ \frac{(t^2+1)^3}{t} } \quad \dots \text{③}
\]
$f(t) = \frac{(t^2+1)^3}{t}$ とおく。$p = t^{\frac{1}{3}}$ とすると $p > 0$ で
\begin{align*}
f(t) &= \left( \frac{p^6+1}{p} \right)^3 = \left( p^5 + \frac{1}{p} \right)^3 = \left( p^5 + 5 \cdot \frac{1}{5p} \right)^3 \\
&\ge \left( 6 \sqrt[6]{ p^5 \cdot \left(\frac{1}{5p}\right)^5 } \right)^3 = 6^3 \cdot \left(\frac{1}{5}\right)^{\frac{5}{2}} \quad (\because \text{AM-GM})
\end{align*}
(等号成立は $p = \left(\frac{1}{5}\right)^{\frac{1}{6}}$ つまり $t = \sqrt{1/5}$ の時)

だから、③に代入して
\[
\min \frac{S(t)}{\sqrt{t}} = \frac{2}{3} \sqrt{ 6^3 \left(\frac{1}{5}\right)^{\frac{5}{2}} } = \frac{4}{5} \sqrt{\frac{6}{\sqrt{5}}}_{\mathbin{/\mkern-6mu/}}
\]
