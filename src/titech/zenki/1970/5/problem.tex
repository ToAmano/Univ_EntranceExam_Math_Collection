\begin{enumerate}
  \item $n$を自然数とするとき$0\leqq\theta\leqq\pi$に対して
不等式$|\sin n\theta| \leqq n\sin\theta$が成り立つことを証明せよ．
  \item 関数$f(\theta)$は次の条件(a)，(b)を満たすとする：
\begin{enumerate}
  \item $f(\theta)\geqq0$
  \item $\displaystyle\int_0^\pi f(\theta)\sin\theta d\theta=1$
\end{enumerate}
このとき，不等式$\displaystyle\int_0^\pi f(\theta)\sin n\theta d\theta \leqq n$が
成り立つことを証明せよ．
\end{enumerate}
