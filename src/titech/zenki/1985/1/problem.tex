2つの条件
\begin{enumerate}
  \item $a^2-2b^2=1$または$a^2-2b^2=-1$
  \item $a+\sqrt2 \, b>0$
\end{enumerate}
を満たす任意の整数$a$，$b$から得られる実数$g=a+\sqrt2 \, b$全体の集合を$G$とする．
1より大きい$G$の元のうち最小のものを$u$とする．
\begin{enumerate}
  \item $u$を求めよ．
  \item 整数$n$と$G$の元$g$に対し，$gu^n$は$G$の元であることを示せ．
  \item $G$の任意の元$g$は適当な整数$m$によって，$g=u^m$と書かれることを示せ．
\end{enumerate}
