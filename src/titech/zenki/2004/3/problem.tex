3枚のコイン$P$，$Q$，$R$がある．
$P$，$Q$，$R$の表の出る確率をそれぞれ$p$，$q$，$r$とする．
このとき次の操作を$n$回繰り返す．
まず，$P$を投げて表が出れば$Q$を，裏が出れば$R$を選ぶ．
次にその選んだコインを投げて，表が出れば赤玉を，裏が出れば白玉をつぼの中にいれる．
\begin{enumerate}
  \item $n$回ともコイン$Q$を選び，つぼの中には$k$個の赤玉が入っている確率を求めよ．
  \item つぼの中が赤玉だけとなる確率を求めよ．
  \item $n=2004$，$\displaystyle p=\frac{1}{2}$，$\displaystyle q=\frac{1}{2}$，$\displaystyle r=\frac{1}{5}$のとき，
つぼの中に何個の赤玉が入っていることがもっとも起こりやすいかを求めよ．
\end{enumerate}
