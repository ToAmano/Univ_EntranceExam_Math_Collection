## 【解】

  (1)
  A, B の手持ちの 2 枚を $a_1, a_2, b_1, b_2$ とする ($a_1 < a_2, b_1 < b_2$)

  この大小関係は以下の${}_4C_2=6$通り考えられる．
  

\begin{enumerate}
[i]
    \item  $a_1 < a_2 < b_1 < b_2$
    \item  $a_1 < b_1 < a_2 < b_2$
    \item  $a_1 < b_1 < b_2 < a_2$
    \item  $b_1 < a_1 < b_2 < a_2$
    \item  $b_1 < a_1 < a_2 < b_2$
    \item  $b_1 < b_2 < a_1 < a_2$
\end{enumerate}

  さて，Bが得点を最大化するような戦略は以下のようなものである．

  《ルール》
  

\begin{itemize}
\item Aが出した札に勝つ札を持っている時, そのうち最も小さい札を出す.
    \item Aが出した札に勝つ札がない時, 最も小さい札を出す.
\end{itemize}

  したがって, たとえば iv の場合, Aが$a_1$ または $a_2$ を 1 枚目に出す時の A の得点は
  

\begin{itemize}
\item $a_1 \to b_2 \to a_2 \to b_1$ で 1 点
    \item $a_2 \to b_1 \to a_1 \to b_2$ で 1 点
\end{itemize}

  のようになる.
  他の場合も同様にして考えると, iからvi での A の得点は 1 枚目に出す札によらず, 以下のようになる.
  

<figure id="1995-3:table:1" class="table-wrapper">

|  各  |  i  | ii  | iii | iv  |  v  | vi  |
|:----:|:---:|:---:|:---:|:---:|:---:|:---:|
| 得点 |  0  |  0  |  1  |  1  |  1  |  2  |

  <figcaption>表 1: 持っているカードと，Aの得点の一覧表</figcaption>
</figure>

  従って，Aはどちらのカードを出しても得点は変わらない．$\cdots$(答)

  

  (2)
  A，Bに無作為にカードを配った場合， 各パターンになる確率は$a_1,a_2,b_1,b_2$の対称性からいずれも等しく$1/6$ずつである．
  従って[表1](#1995-3:table:1)より
  Aの得点の期待値は
  

$$
\begin{align*}
E
     & = \frac{0}{6} + \frac{0}{6} + \frac{1}{6} + \frac{1}{6} + \frac{1}{6} + \frac{2}{6}\\& = \frac{5}{6}
\end{align*}
$$

  である．$\cdots$(答)

  
  (3)
  $a_1+a_2=14$のとき，$(a_1, a_2)$ とそれに対応する iからvi のパターンとなる場合の数は[表2](#1995-3:table:2)のようになる．

  したがって [表1](#1995-3:table:1) の得点表とあわせて,
  各々のパターンの時のAの得点の期待値は[表3](#1995-3:table:3)のようになる．
  ここで，$b_1, b_2$ のえらび方は $_{13}C_2$ で同様に確からしいことに注意する．

  

<div id="tab_3" class="table-wrapper">

  したがって，求める期待値の最大最小は
  

\begin{itemize}
\item $(4, 10)$ の時. $E = 1 - \frac{15}{_{11}C_2} = \frac{8}{11}$
    \item $(1, 12)$ の時. $E = 1$
\end{itemize}

  である．$\cdots$(答)
