$n$を自然数とする．
次の3つの不等式(1)，(2)，(3)を全て満たす自然数の組$(a,b,c,d)$はいくつあるか．
$n$を用いてあらわせ．

\begin{itemize}
\item $1\le a < d \le n$
  \item $a\le b < d$
  \item $a<c\le d$
\end{itemize}
